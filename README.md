# Hermes Agent 知识库

整理自微信公众号文章（2026年4月）

## 在线访问

部署后访问：`https://<your-username>.github.io/knowledge/`

## 本地开发

```bash
npm install
npm run dev
```

## 部署到 Cloudflare Pages

1. Fork 或上传到 GitHub 仓库
2. 登录 [Cloudflare Dashboard](https://dash.cloudflare.com/)
3. Workers & Pages → Create application → Pages → Connect to Git
4. 选择仓库，Build command: `npm run build`，Output directory: `.vitepress/dist`
5. Deploy
